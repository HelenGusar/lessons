My practice: 2D-transform

Що вміє...

...translate?

Зміщує елемент на нове місце відносно початкового положення вправо та вниз через координати X та Y.

...scale?

Масштабує елементи через координати X та Y. Значення від 0 до 1, де 1 — цілий обʼєкт. Значення за замовчуванням для `scale` - `transform: scale(1);`

Масштабує елементи: робить більшими. Масштабує елементи через координати X та Y. Значення від 0 до 1, де 1 — цілий обʼєкт.

Масштабує елементи: робить меньшими. Масштабує елементи через координати X та Y. Значення від 0 до 1, де 1 — цілий обʼєкт.

...rotate?

Повертає елементи, додатні значення повертають елемент по годинниковій стрілці, від’ємні проти годинникової стрілки.

Обертання по осi “х” виглядає наче як ви обертаєте листок паперу по осі “x”

Обертання по осi “y” виглядає наче як ви обертаєте листок паперу по осі “y”

...skew?

Дозволяє нахилити елемент в обох напрямках, змінюючи його геометричний вигляд та створюючи вигнуту площину нахилу.

Для використання skew() вам потрібно передати два аргументи: перше значення вказує нахил в горизонтальному напрямку, а друге - в вертикальному напрямку.

My practice: 3D-transform

Що вміє...

...translate3D?

Переміщення елемента в 3D-просторі. Крім осі X та Y додається вісь Z.

...scale3D?

Масштабує елементи через координати X,Y,Z. Значення від 0 до 1, де 1 — цілий обʼєкт.

...rotate3D?

Повертає елементи, додатні значення повертають елемент по годинниковій стрілці, від’ємні проти годинникової стрілки.

...transform-style?

Визначає, як дочірні елементи мають відображатися у 3D-просторі. Задається батьківському елементу разом з `perspective`.
